Agents must only select modules relevant to the job and customise controls to site conditions, client requirements, permits and plant. ## Scope Applies to TMC enviro / MCM Services operations: civil, vegetation, road, rail, contaminated soils and related activities. ## Risk Matrix (IMS-aligned) Likelihood (1–5) × Consequence (1–5) = Risk - Low: 1–4 - Medium: 5–9 - High: 10–14 - Extreme: 15–25 Note: R ≥ 10 = significant. ## Hierarchy of Controls 1. Eliminate 2. Substitute 3. Isolate / Engineer 4. Administrative Controls 5. PPE ## Agent Selection Rules (v0.3) 1. Determine HRCW triggers first. 2. Select only modules relevant to scope, plant, location, client and work sequence. 3. If multiple modules apply, include all applicable modules. 4. Do not include confined space unless specifically identified. 5. Follow the hierarchy of controls; do not default to admin/PPE only. 6. Flag missing permits, service plans, TMPs, asbestos plans and client hold points. 7. Use IMS risk badge language: EXTREME (15–25), HIGH (10–14), MED (5–9), LOW (1–4). 8. Asbestos work — licensed subcontractor controls apply unless TMC/MCM confirmed to perform the work. 9. Include assumptions and review hold points rather than invent facts if information is missing. 10. Do not invent IMS SWI/CP numbers; flag DMS-validation required if uncertain. 11. Residual HIGH/EXTREME = supervisor/manager review required. 12. External client/authority rules (e.g. ARTC, V/Line, John Holland) must be added as external controls — they do not replace IMS controls. ## How to use this document - Use this MD as the source of truth for agent module selection. - Use the full HTML (landscape) for presentation and printing. - For SWMS generation: select applicable modules, copy into SWMS template, customise per site, and run DMS reference validation. ## Module Index (HM-001 → HM-031) -
